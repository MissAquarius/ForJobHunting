## 文件目录命令
* ls
  * 功能：列出目标目录中所有的子目录和文件
  * 参数：
    * -a 所有文件
    * -l 列出文件的详细信息，包括文件名、权限、所有者、文件大小等信息
    * -R 同时列出所有子目录层
    * -h 以容易理解的格式列出文件大小
* cd
  * 功能：切换目录
  * 进入 系统主目录：cd / 
  * 进入 当前用户根目录：cd 或者 cd~
  * 返回进入此目录之前所在的目录： cd -
* pwd
  * 查看当前所在目录
  
* mkdir
  * 功能：创建目录，要求创建目录的用户在当前目录中具有写权限，并且指定的目录名不能是当前目录中已有的
  * 参数：
    * -m 设定权限（mkdir -m 777 test）
    * -p 递归创建多个目录
    * -v 显示创建信息
* rm
  * 功能：删除文件或目录（-r）
  * 参数：
    * -f 强制删除，不需确认（删除以）
    * -i 交互式删除
    * -r或-R 递归删除当前目录及所有子目录
* rmdir
  * 功能： 删除空目录
